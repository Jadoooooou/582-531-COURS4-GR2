## Pour importer un VST:   
TÉLÉCHARGER LE DOSSIER Wavetable.dll    
DANS Option -> Preferences -> VST -> Edith path list -> Re-scan   
ENSUITE, DOUBLE-CLICK POUR CRÉER UNE PISTE -> FX -> VSTi -> VSTi: Wavetable SocaLabs
